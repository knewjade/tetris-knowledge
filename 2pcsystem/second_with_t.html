パフェ2回目：Tあり¶

基本的に3地形でパフェ1回目テンプレの残りミノはカバーできます

「TIOS = TIOZ」「OTSZ」はこの3地形で対応できませんが、 パフェ1回目にテンプレを使用した場合、このパターンは残りません。

グレースシステムを組む方は、個別で対応する必要あります（補足 を参考にしてください）

対応表¶

取り消し線 は グレースシステム＆テンプレ で残らない組み合わせです

TIO* A B C
TIOL = TIOJ   O  
TIOS = TIOZ      
IT** A B C
ITLJ   O O
ITSZ      
ITLS = ITJZ O O O
ITLZ = ITJS O O O
OT** A B C
OTLJ      
OTSZ      
OTLS = OTJZ     O
OTLZ = OTJS   O O
T*** A B C
TLJS = TLJZ   O  
TLSZ = TJSZ O    

注釈

基本地形は ふちさん にまとめていただきました。ありがとうございます。 https://twitter.com/kitsune_fuchi

T と [SZ] のおく場所を固定するパターンです。

OTSZ¶


最初に置いたOの上の2ラインの埋め方が JI (LI) と OL (OJ) があるので、他のTありパターンと比べると少し難しい。

基本的な考えは JI (LI) で組めるか確認して、できないときはOL (OJ)を入れます。

セットアップ：テト譜

消しかた：テト譜
